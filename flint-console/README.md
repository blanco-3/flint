# Flint Console

Flint Console is the visible product shell for Flint's protected execution backend.

## Modes

- **Seeded Judge Mode**
  - renders tracked benchmark and devnet smoke artifacts
  - works even if the relay is offline
- **Live Relay Mode**
  - talks to the local relay at `http://127.0.0.1:8787`
  - creates quote requests and monitors request status
